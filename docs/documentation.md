# Documentation

This document outlines the requirements for creating a new product.

## Required Fields

- **Collection**: Required when creating a new product and is used as Subcategory (multiple allowed).
- **Type**: Must be one of E-Bike, E-Cargobike, Kinderrad which is then mapped by Smart Collections.

## Optional Fields

- **Description**: (optional) A detailed description of the product.
- **Images**: (optional) Upload one or more images of the product.

## Pricing

- **Price**: The selling price of the product.
- **Sale Price**: The price of the product during a sale, if applicable.

## Inventory

- **Stock Keeping Unit (SKU)**: Unique identifier for each distinct product and service that can be purchased.
- **Inventory Policy**: Indicates whether customers can purchase an out-of-stock item.

## Shipping

- **Weight**: The weight of the product for shipping purposes.
- **Dimensions**: The dimensions of the product (length, width, height).

## Tags

- **Tags**: Add tags related to the product to help with search and categorization.

## Variants

- **Variants**: Options for the product that can vary such as size, color, etc.

Please ensure that all required fields are filled out correctly to avoid any issues with the product listing.
